Device Trust Setup

1 Enable Developer Options

Open Android Settings, go to About phone, then tap Build number seven times to unlock developer controls.

2 Turn On USB Debugging

Enable USB debugging so DroidSentinel can perform a local ADB security audit.

3 Connect And Allow

Connect by USB, unlock the phone, and authorize the RSA debugging prompt for this computer.

Connection Troubleshooting

Use this checklist when DroidSentinel cannot audit the device.

Unlock the phone and accept the USB debugging prompt.

Use a USB data cable, not a charge-only cable.

Switch USB mode to File transfer if Windows only sees charging.

Install the manufacturer USB driver if Windows does not detect the phone.

Disable and re-enable USB debugging to trigger a new authorization prompt.
